How we protect your data in our testing environment

Introduction

This privacy policy explains how our test website handles data for the purpose of testing Skardu's multi-page snapshot generation system. Since this is a testing website, no real user data is collected or processed.

Information We Collect

Testing Data

  • Page visit information for snapshot testing
  • User agent strings for mobile-first optimization testing
  • Basic analytics for testing performance

What We Don't Collect

  • Personal information (since this is a test site)
  • Payment data (no real transactions)
  • Sensitive personal data

How We Use Information

Any data collected is used solely for:

  • Testing Skardu's multi-page snapshot generation
  • Verifying mobile-first optimization
  • Ensuring proper SEO functionality
  • Website performance analysis

Data Security

Even as a test website, we implement standard security measures:

  • HTTPS encryption for all pages
  • Secure hosting environment
  • Regular security updates
  • No storage of sensitive information
